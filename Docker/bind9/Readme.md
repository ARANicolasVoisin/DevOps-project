# Ubunu distribution

Si vous être sur une distribution Ubuntu, vous devez décommenter la ligne **DNSStubListener** dans le fichier **/etc/systemd/resolved.conf**. Autrement vous aurez un message d'erreur.

If you're running the container on Ubuntu disrtibution, you must uncomment the line **DNSStubListener** in the file **/etc/systemd/resolved.conf**. Otherwise you have an error message.
